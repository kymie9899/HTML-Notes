Kim's HTML Coding Notes

Command + / ( also <strong> )

Creates the line the cursor is in or any text highlighted into a comment note

type html and hit tab button (also <em>)

populates the html boilerplate

type your code like "ul" and push tab ( also <ins> )

creates the correct open and closing brackets for the tag

command + shift + D ( also <sup> )

copies the line above the cursor and duplicates it below

type lorem and hit tab ( also <sub> )

creates a paragraph of lorem ipsum text for placeholder

command + click in multiple lines and areas in your file ( also <del> )

the cursor is in all the places you clicked. Then start typing and it types the same thing in every place. so ie. <img width="50" can be added to all the images lines at the same time.
